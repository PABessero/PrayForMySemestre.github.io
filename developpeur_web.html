Développeur WEB

  • Présentation du métier : Un développeur s'occupe de créer un site Internet en fonction du cahier des charges donné par le client. Il va donc rédiger des lignes de codes pour obtenir le résultat escompté.
  • Missions :
    • Utilisation du HTML;
    • Utilisation du CSS;
    • Utilisation du JavaScript;
    • Utilisation du PhP / MySQL
  • Compétences : Savoir utiliser tous les langages informatique.
  • Formation : BTS (Info, SIO), DUT Info, École d'ingénieur, Licence Pro, Master Pro.
  • Salaire : Début : 40.000 € par an Fin : 50.000 € par an.
